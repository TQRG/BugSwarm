Resources available API resources

OIPA currently implements five public resources:

  • OrganisationResource *
  • ActivityResource
  • RegionResource
  • CountryResource
  • SectorResource
  • ActivityFilterOptionsResource (JSON only)
  • IndicatorFilterOptionsResource (JSON only)
  • IndicatorDataResource (JSON only)
  • CountryGeoJSONResource (JSON only)
* The current API implementation nests basic OrganisationResource fields within the ActivityResource as well as Region-, Country- and SectorResources.

ActivityFilterOptionsResource (JSON only)

Filtering options based on country, region, sector, budget and organisation filters. Example: when using parameters: organisation = NL-1, all countries, regions and sectors are returned this organisation has activities in.

Available fields:

Field Example
regions regions: {
89: "Europe, regional",
189: "North of Sahara, regional",
289: "South of Sahara, regional"
}
sectors sectors: {
11110: "Education policy and administrative management",
11120: "Education facilities and training"
}
countries countries: {
BD: "BANGLADESH",
BF: "BURKINA FASO"
}

O pré-requisito para o envio de pacotes por FTP é ter um cliente de FTP instado. Um cliente grátis e multi-plataforma é o FileZilla.

Para enviar os pacotes por FTP siga os seguintes passos:

  1. Connectar ao servidor de FTP ftp://roda.dgarq.gov.pt;
  2. Carregar o conteúdo de %1$s para a sua pasta de entrada;

Importar de Microsoft Access

Para importar bases de dados em Microsoft Access são necessários alguns passos manuais. A aplicação destes passos é rápida e simples, e pressupõe a existência de um Microsoft Access instalado na máquina onde será feita a importação pelo RODA-in.


Activar opção para ver objectos de sistema

É necessário activar a opção de visualização de objectos de sistema para conseguir efectuar os passos seguintes. Este passo só precisa de ser efectuado uma vez numa mesma instalação do Microsoft Access:

  1. Aceda às opções do Microsoft Access;


  2. No separador View, em Show, escolha System Objects;


  3. Carregue em OK.

Adicione permissões de leitura ao Catálogo Microsoft

Para cada base de dados a importar, é necessário adicionar permissões de leitura ao Cátalogo Microsoft, para que as informações de estruturas possam ser extraídas:

  1. Aceda às permissões de Utilizadores e Grupos;


  2. No separador Permissions, na lista Object Name, escolha a tabela MSysRelationships;


  3. Active as permissões Read Design e Read Data;
  4. Carregue em OK.

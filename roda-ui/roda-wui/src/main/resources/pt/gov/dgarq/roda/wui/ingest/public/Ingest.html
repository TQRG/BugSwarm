This is a static page in RODA, in particular, Ingest.html.

In order to modify the contents of this file, one just need to edit file [RODA_HOME]/webapps/roda-wui.war/Ingest.html

Also, if one wants to have this page in multiple languages, one just needs to create one file per language. For example, to have english support create a file called Ingest_en.html, for portuguese (from Portugal) create a file called Ingest_pt_PT.html

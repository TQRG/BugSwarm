This program allows the creation of submission information packages (SIP) to be integrated on the RODA digital archive. This packages follow the OAIS and the RODA principles. For the integration to be complete, it is necessary that the created SIPs are sent on any conventional optical support via the postal mail to the address:

 Alameda da Universidade
 1649-010 LISBOA, PORTUGAL
 +351 21 781 15 00
 roda@dgarq.gov.pt

RODA is a digital archive for digital object created by the public administration and it is a service of the Portuguese National Archives.

This program was developed by the Portuguese National Archives with the colaboration of the University of Minho. The project was developed between 2006 and 2008 and was partially supported by the Public Administration Operational Program (POAP)

More information can be found at http://roda.dgarq.gov.pt

DISCLAMER:

Not every digital object can be processed by this program. For more information about the use limits of this program visit http://www.dgarq.gov.pt/roda.

======
f90nml
======

SYNPOSIS
========

f90nml
   [-h] [--version] [--group GROUP] [--variable VARIABLE] [--patch]
   [--format FORMAT] [--output OUTPUT] [input] [output]


DESCRIPTION
===========

``f90nml`` is a command line tool for the reading, writing, and the general
manipulation of Fortran namelist files.

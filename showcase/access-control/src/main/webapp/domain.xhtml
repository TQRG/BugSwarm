Domain demo: you have access! You accessed this page at the domain: #{facesContext.externalContext.requestServerName}

This is a domain-based access control demo. Access is granted when this page is requested by the domain "localhost" or by any domain deployed to Red Hat's Openshift Express cloud. (See the demo below) for info on how to get started deploying apps to the cloud in about 10 minutes.)

If you want to see the Access Denied page. Try accessing this page locally at 127.0.0.1, or use the IP address of the cloud server instead of the qualified DNS name.

Domain-based access control can be useful providing different types of access for different domain users, like what github does. Crazy!

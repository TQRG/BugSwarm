Timer demo: you have access! But access will be denied #{timerBean.secondsUntilDenied}

This is a time-based access control demo. Access is granted for the first 30 seconds of each minute (1-30); subsequently, access is denied for the final 30 seconds of each minute (31-60).

Time-based access control can be useful for providing scheduled down-time or maintenance windows. Or for crazy things like limiting a user's session for a set period of time, then pulling the plugin on them. Crazy!
